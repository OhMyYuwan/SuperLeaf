# Code of Conduct

YuwanLabWriter should be a respectful and productive space for people working on academic writing tools, collaboration systems, and AI-assisted editing.

## Expected Behavior

- Be kind, patient, and specific.
- Assume good intent, especially across language and cultural differences.
- Give feedback about work, not about people.
- Respect boundaries around privacy, unpublished research, credentials, and project data.
- Prefer clear technical reasoning over status, volume, or pressure.

## Unacceptable Behavior

- Harassment, threats, insults, or targeted personal attacks.
- Discriminatory language or conduct.
- Publishing private information without permission.
- Repeatedly derailing discussions after maintainers ask you to stop.
- Abusing issues, discussions, pull requests, or support channels for spam or unrelated promotion.

## Reporting

If you see behavior that violates this code, contact the maintainers through a private GitHub Security Advisory or another private channel listed by the project owner. If no private channel is available, open a minimal public issue asking for a maintainer contact method without disclosing sensitive details.

Maintainers may remove comments, close issues, restrict participation, or block users when needed to protect the project community.

## Scope

This code applies to project spaces including GitHub issues, pull requests, discussions, documentation channels, and project-related conversations.
